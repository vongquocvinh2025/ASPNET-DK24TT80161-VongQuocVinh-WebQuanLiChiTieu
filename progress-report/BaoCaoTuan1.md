## Báo cáo tiến độ Tuần 1

*(Thời gian: 01/06 – 07/06)*

* **Khởi tạo dự án**:
    * Đã tạo thành công Repository trên GitHub theo đúng cú pháp.
    * Đã thêm thầy hướng dẫn làm Collaborator.
    * Đã cài đặt xong công cụ lập trình Visual Studio, .NET SDK và SQL Server trên máy tính.

* **Phân tích yêu cầu đề tài**:
    * Xác định mục tiêu và phạm vi của hệ thống "Quản lý chi tiêu cá nhân".
    * Liệt kê các chức năng chính: quản lý chi tiêu, thu nhập, danh mục, hạn mức, mục tiêu tiết kiệm, thống kê và nhắc nhở.
    * Xác định hai nhóm tác nhân chính: Người dùng và Quản trị viên (Admin).

* **Thiết kế sơ bộ**:
    * Phác thảo sơ đồ use case tổng quát cho toàn hệ thống.
    * Thiết kế sơ bộ cơ sở dữ liệu và sơ đồ thực thể quan hệ (ERD).
    * Lựa chọn công nghệ: ASP.NET Core MVC, Entity Framework Core và SQL Server.

* **Công việc còn lại**:
    * Khởi tạo cấu trúc dự án ASP.NET Core MVC.
    * Bắt đầu xây dựng các chức năng nền tảng (đăng ký, đăng nhập).
